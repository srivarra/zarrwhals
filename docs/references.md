# References

## Related Projects

- [Zarr](https://zarr.dev/) - Chunked, compressed N-dimensional arrays
- [Narwhals](https://narwhals-dev.github.io/narwhals/) - DataFrame-agnostic interface
- [AnnData](https://anndata.readthedocs.io/) - Annotated data for single-cell analysis
- [Polars](https://pola.rs/) - Fast DataFrame library
- [Dask](https://www.dask.org/) - Parallel computing with task scheduling
